Installation
---
```
git clone https://github.com/YashGoti/Subdomain
cd Subdomain
chmod +x subdomain.sh
```

Usage
---
```
./subdomain.sh domain
```
 
Make subdomain.sh universal
---
```
cp <PATH TO FILE>/subdomain.sh /usr/bin/subdomain
subdomain domain

[Mostly work in Linux Systems]
```
